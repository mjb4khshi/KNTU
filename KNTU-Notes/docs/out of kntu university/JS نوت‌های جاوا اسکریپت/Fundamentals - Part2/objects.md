# شیء در جاوا اسکریپت
شیء رو می‌شه یه جورایی گروهی از متغیرها دونست که به یک گروه نسبت داده شدن.
آرایه‌ها در نگاه عمیق‌تر نوعی شیء به حساب میان
## تعریف شی
```js
const MJ = {
firstName: `mohammadjavad`,
lastName: `bakhshi`,
birthYear: 2005
}
```
اساسا هر مقداری رو میتوان با یک پراپرتی تعریف کرد حتی میتوان آرایه ها رو هم تعریف کرد درون یک شیء.(پراپرتی مثل last name یا الباقی)
## دسترسی به یک پراپرتی در شیء
دو راه برای دسترسی به پراپرتی های درون شیء وجود دارد
### dot notation
تنها کافیست که اسم شی را نوشته و با یک نقطه در جلویش نام پراپرتی را بنویسیم(مانند طول آرایه مثلا)
```js
const MJ = {
firstName: `mohammadjavad`,
lastName: `bakhshi`,
birthYear: 2005
}

console.log(MJ.birthYear); //2005
```
### bracket notation
```js
const MJ = {
firstName: `mohammadjavad`,
lastName: `bakhshi`,
birthYear: 2005
}

console.log(MJ['birthYear']); //2005
```
**تفاوت نقطه و براکت در اینه که واسه نقطه نمیتوان عبارت جلوش گذاشت ارور میده در حالی که توی براکت میشه عبارت گذاشت
مثل آرایه ها، پس جایی که عبارت خواستی بذاری از براکت و جاهای دیگه میتونی از نقطه استفاده کنی برای سادگی و خوانایی**
