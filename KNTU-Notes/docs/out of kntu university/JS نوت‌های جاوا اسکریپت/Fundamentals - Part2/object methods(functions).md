# متدهای شی در جاوا اسکریپت
همانطور که توی آرایه ها(که در واقع خودشون نوعی شی هستن)ما متد ها و توابعی برای اعمال کار داشتیم، در اشیا هم همینطوره ولی باید توباع رو خودمون درون شی تعریف کنیم و بعدا فراخوانی کنیم چه در خود شی چه بیرون شی.
```js
const mark = {
  fullName: `Mark Miller`,
  mass: 78,
  height: 1.69,
  calcBMI: function () {
    return this.mass / (this.height * this.height);
  },
};
```
در اینجا ما تابع محاسبه بی ام آی رو نوشتیم و برای فراخوانی کافیه مثل متدهای آرایه آن را فراخوانی کرد.
```js
mark.calcBMI();
```
**حتما حواست باشه برای فراخوانی جلوی اسم تابع () رو بذاری و حتی اگه آرگومان هم نداشتی باید بذاری و الا رفرنس میزنه و نتیجه درست رو نمیده در واقع فراخوانی نمیشه**
#methods
برای تظیم یک پراپرتی هم کافیه که مقدار آن را با عملگر انتسابی تغییر دهیم یا اضافه کنیم
مثلا:
```js
mark.wife = `angela`;
```
### متدهای جاوا اسکریپت باید به شکل عبارت تعریف شن و نمیشه با فلش یا حالات دیگه تعریف کردشون
