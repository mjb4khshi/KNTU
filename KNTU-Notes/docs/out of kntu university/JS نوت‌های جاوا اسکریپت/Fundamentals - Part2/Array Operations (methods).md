# عملگر های آرایه(متد)
عملگرهایی هستن که روی آرایه ها اعمال میشن و میشه با اون‌ها آرایه‌ها رو ویرایش کرد.
## عملگر push 
این متد در واقع یک عضو یا عنصر به آخر آرایه اضافه میکنه
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
friends.push('ali'); //['mahdi', 'ilia', 'amirreza', 'ali']
```
از اونجایی که توابع هرکدوم میتونن یه مقداری هم برگردونن و عملگرهای آرایه در واقع از جنس توباع هستن، اینجا مقداری که تابع push برمیگردونه، تعداد عناصر آرایه در حالت جدیده که البته خب خیلی به کار نمیاد ولی میشه مقدارشو ذخیره کرد.
## عملگر unshift 
این متد در واقع برای اضافه کردن یه عنصر به اول آرایه(جایگاه ۰) هستش.
و مقداری که برمی‌گردونه هم باز مثل بالا تعداد عناصر آرایه در حالت جدیده
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
friends.unshift('ali'); //['ali', 'mahdi', 'ilia', 'amirreza']
```
## عملگر pop
این متد برخلاف بالایی یه عنصر از آخر آرایه حذف میکنه
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
friends.pop(); //['mahdi', 'ilia']
``` 
و مقداری که بر میگردونه در واقع عنصری هست که حذف شده.
## عملگر shift
این متد از اول آرایه یه عنصر رو جذف میکنه و مقداری هم که برمیگردونه در واقع همون آرایه حذف شده خواهد بود.
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
friends.shift(); //['ilia', 'amirreza']
``` 
## نکته بسیار مهم راجع به const
**در همه مثال های بالا دیدی که آرایه ها با اینکه با const تعریف شدن ولی تونستیم ویرایشش کنیم، اون ثابت بودن واسه مقادیر بدویه که اشاره کردیم قبل تر مثل عدد و رشته، در مورد آرایه فقط خود آرایه نمیتونه بازتعریف شه و الا میشه روش اعمال و متد ها رو اعمال کرد.

## عملگر indexof 
در واقع برای جستجوی یک مقدار در آرایه میشه ازش استفاده کرد و در صورت موجود بودن شماره عنصر(با پایه صفر منطقا) برمیگردونه
و اگه موجود نباشه -1 بر میگردونه.
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
console.log (friends.indexof('mahdi')); // 0
console.log (friends.indexof('hossein')); // -1
``` 
## عملگر includes
این عملگر مدرن‌تره و واسه بررسی وجود یا عدم وجود یک عنصر در آرایه است و مقداری که برمی‌گردونه بولین هستش.
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
console.log (friends.includes('mahdi')); // true
console.log (friends.includes('hossein')); // false
```
