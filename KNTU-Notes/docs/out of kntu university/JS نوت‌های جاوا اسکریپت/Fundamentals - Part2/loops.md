# حلقه ها
## for
```js
for (let index = 0; index < 10 ; index++) {
  console.log(`چاپ این رشته برای بار ${index+1} ام`)
}
```
بخش اول ساختار کنترلی ما به متغیری مربوط میشه که قراره طی حلقه عوض شه و باهاش کنترل کنیم دفعات رو
بخش دوم شرطیه که قبل هربار رخ دادن حلقه بررسی میکنیم
و بخش سوم اتفاقیه که بعد هر بار رخ دادن حلقه انجام میشه
برای مثال خروجی این تیکه کد چاپ 10 بار رشته با عوض شدن عدد از 1 تا 10 است.
### break
هنگامی که این رو بذاریم در جا از حلقه میپره بیرون
### continue
فقط همون یه دفعه از حلقه رو رد میکنه و میره سراغ سری بعدی حلقه و از حلقه در واقع نمیپره بیرون مثل برک
# while
ساختارش اینطوریه که تا زمانی که شرطی که بهش میدیم درست باشه اجرا میشه
واسه وقتایی کاربردیه که شمارندمون ترتیبی نمیره یا اصلا نمیدونیم طولش چقدره و چند دفعه میخوایم تکرار شه یا قراره تکرار شه
```js
while (1>10){
console.log(`سبجان الله`)}
```
حلقه اجرا نمیشه اینجا مثلا چون خب واضحا جواب شرط فالسه.
