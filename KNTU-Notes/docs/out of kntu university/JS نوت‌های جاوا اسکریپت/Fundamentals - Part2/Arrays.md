# آرایه ها
آرایه ها اساسا یک نوع غیر بدوی از داده ها هستند که میتوان چندین داده را در آنجا ذخیره و سپس به سادگی به تمامی آنها دسترسی پیدا کرد.
## تعریف آرایه:
```js
const friends = ['mahdi', 'ilia', 'amirreza'];
```
یا 
```js
const friends = new array('mahdi', 'ilia', 'amirreza');
```
که البته روش اول خواناتره و توصیه میشه
## دسترسی به یک مقدار درون آرایه:
برای مثال ما میخواهیم که به داده دوم آرایه دسترسی پیدا کنیم 
کافیه به شکل زیر عمل کنیم **توجه کن که آرایه ها پایه صفر دارن یعنی اولین عنصر آرایه در واقع عنصر 0 به حساب میاد**
```js
//آرایه قبلا تعریف شده بالا
console.log(friends[1]); //ilia
```
## بررسی طول یک آرایه:
برای بررسی طول یک آرایه به سادگی تنها کافیست که به شکل زیر عمل کنیم:
```js
console.log(friends.length); //3 طول در واقع پایه صفر نداره و تعداد عناصر رو نشون میده
```
## دسترسی به عنصر آخر آرایه:
برای دسترسی به عنصر آخر کافیه که طول آرایه رو منهای یک کرده و در براکت قرار بدیم
در واقع توی براکت لزوما نیاز نیست عدد باشه میتونه عبارت باشه!
## آرایه های چند بعدی
میتوان درون یک آرایه آرایه ای دیگر تعریف کرد
مثال:
```js
const MJ = ["mj", "bakhshi", ["ali", "mahdi"]];
console.log(MJ[2][1]) //mahdi
```
و همونطور که دیدی برای دسترسی به عناصر داخل آرایه درونی باید بعد ایندکس اول ایندکس عنصر آرایه درونی رو بنویسی
