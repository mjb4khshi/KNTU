# عبارت شرطی
یه چیز خیلی کاربردی و خفن جی اس مدرنه که اجازه میده توی یه خط یه قالب شرطی جم و جور انجام بدیم
در واقع خروجیش یه مقداره و این یه عبارت به حساب میاد
مثال و نوع استفاده:
```js
const age = 18;
const isAdult = age>=18 ? true : false //اینجا بررسی میکنه اگه جواب شرط درست بود ارزش اولی و اگه نبود ارزش دومی رو بر میگردونه
```
قسمت اول شرط
قسمت دوم در واقع اگر شرط درست بود
قسمت سوم اگه شرط اشتباه بود
طبق الگو
