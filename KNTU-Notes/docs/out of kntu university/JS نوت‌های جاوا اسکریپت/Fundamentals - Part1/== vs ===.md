# عملگرهای مقایسه تساوی
برای بررسی تساوی دو مقدار ما باید از عملگرهای == و === استفاده کنیم.(حتما به خاطر داری که = در واقع عملگری برای انتساب مقداره و اصلا مقایسه ای نیست)
در == ما بررسی صرف مقادیر رو داریم و به اون مقایسه شل هم میگن و در واقع اگر یک رشته با عدد را مقایسه کنیم ابتدا رشته را به شکل ضمنی به عدد تبدیل میکند و سپس مقایسه انجام می‌شود. مثال زیر:
```js
const age='18';
age == 18;//true
```
ولی در مورد === اینطوری نیست و حتما باید مقادیر نوع یا تایپ یکسانی داشته باشند.
مثال زیر:
```js
const age='18';
age === 18;//false
```
**طبق اصول کلین کد و برای جلوگیری از بروز خطا تا جای ممکن از == استفاده نشه و اگر نیاز به تبدیل نوع داشتیم به صورت دستی با توابع خودش تبدیل کنیم و اصلا فکر کن == وجود نداره.**
